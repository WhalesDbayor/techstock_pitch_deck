Technology

Built on a Modern, Scalable Stack

React 18 (Vite) Node.js / Express 5 MongoDB / Mongoose JWT Auth Socket.io IndexedDB (Offline) Multer (Image Upload) CSV Import/Export Vercel + Render
Security-first — bcrypt password hashing, JWT middleware, role-based route guards on every endpoint
Offline-first PWA — IndexedDB queue replays failed requests automatically when connection restores
Real-time — Socket.io delivers live low-stock alerts to all connected admin & staff sessions
Architecture Highlights
  • 13 backend data models (Products, Orders, Customers, Suppliers, Returns, Debts, Expenses, Notifications…)
  • 50+ REST API endpoints with filter, pagination & export
  • Granular permission middleware on all protected routes
  • CSV import/export via json2csv & csv-parser
  • Seed script with realistic sample data for demos
  • Dark & light mode via CSS custom properties + React Context
Deployment
Frontend on Vercel, backend on Render — both currently on free tiers during early traction phase. Production-grade infrastructure ready to upgrade on revenue.
